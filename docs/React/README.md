## 🧠 Concepts Clés
- **Virtual DOM** : Performance et rendu efficace.
- **Unidirectional Data Flow** : Les données descendent via les props.
- **Hooks** : Gestion de l'état et des cycles de vie dans les composants fonctionnels.
- **TypeScript** : Sécurité et robustesse du code frontend.
