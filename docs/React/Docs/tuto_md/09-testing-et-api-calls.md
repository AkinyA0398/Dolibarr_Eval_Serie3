# Testing et Appels API

## Testing

- **Vitest** + **React Testing Library** (recommandé)
- Jest (encore très utilisé)
- Cypress / Playwright (tests E2E)

## Appels API

### REST
- **Axios**
- **TanStack Query (React Query)** → **fortement recommandé**

### GraphQL
- Apollo Client, urql, Relay

**Meilleure pratique actuelle** : `TanStack Query` + Axios ou Fetch.
