# Introduction à React - Roadmap Complète

## Prérequis obligatoires

Avant de commencer React, tu **dois** maîtriser les sujets "Beginner" du **JavaScript Roadmap** :
- Variables, fonctions, objets, arrays
- Arrow functions, destructuring, spread/rest
- Modules (import/export)
- Async/Await + Promises
- DOM manipulation basique

**Outil recommandé pour créer un projet React :**
- **Vite** (recommandé) : `npm create vite@latest mon-projet -- --template react-ts`

---

## Objectif de cette série

Apprendre React de manière **progressive** :
- **Niveau Débutant** : Bases solides
- **Niveau Intermédiaire** : Hooks avancés + architecture
- **Niveau Avancé** : Optimisations, patterns complexes, écosystème

**Note** : Les Class Components sont **dépréciés**. On se concentre uniquement sur **Functional Components + Hooks**.
