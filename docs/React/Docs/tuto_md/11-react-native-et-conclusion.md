# React Native et Conclusion

## React Native

Pour créer des applications mobiles avec le même langage (React).

**Apprendre après avoir bien maîtrisé React Web.**

## Frameworks recommandés

- **Next.js** (le plus populaire)
- **Astro** (pour sites statiques/performants)
- TanStack Router (alternative moderne)

## Ordre conseillé

1. Bases + Hooks
2. Routing + Forms + Styling (Tailwind + Shadcn)
3. Gestion d’état (Zustand / TanStack Query)
4. Testing
5. Next.js

**Ressource officielle** : [roadmap.sh/react](https://roadmap.sh/react)
